Movie API Documentation

This is a movie API, which contains movie title, description, genre, director, imageURL.

API Endpoints
Requests URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data of all the movies.
Get all information about one sigle movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing title, description, genre, director, imageURL and feutured or not. Example:
{ "Title":"The Fellowship of the Ring", "Description": "Teleplay based on the of J.R.R. Tolkien's novel 'The Fellowship of the Ring'.", "Genre": { "Name": "Fantasy", "Description": "Description1" }, "Director": { "Name": "Peter Jackson", "Bio": "Bio PJ", "Birth": "Birth PJ" }, "imageURL":"https://m.media-amazon.com/images/M/MV5BZTQ4YTA1YmEtNWY1Yy00ODA2LWI2MGYtZGY2ZTgzYjEzMDZjXkEyXkFqcGdeQXVyNTE1MDE2MzY@._V1_SX300.jpg" }
Get data about a genre by genre name /movies/genre/[Name] GET None A JSON object holding data about a single genre, containing name and description.
Get data about a director by name /movies/directors/[Name] GET None A JSON object holding data about a single director, containing name, bio and birth year.
Creat a new user /users POST A JSON object holding data about the new user, containing name, id and favoriteMovies. Example:
{"name":"Tom Hanks","favoriteMovies": []}
 A text message indicating whether the user has been registered.
Update a user's name /users/[ID] PUT None A text message indicating the user's name has been updated.
Update a user's favoriteMovies /users/[ID]/favmovies POST A JSON object holding data about a new favorite movie of the user, containing title. Example:
{"Title": "Harry Potter"}
 A text message indicating the movie has been added to the favorite moive list.
Delete a movie from the favoriteMovies /users/[ID]/favmovies/[Title] DELETE None A text message indicating a movie has been deleted from the favorite movie list.
Delete an user's data /users/[ID] DELETE None A text message indicating the user's data has been removed.
